BetterEdit

    HJfod
    Editor
    Popular

    Description

    The ultimate editor improvement mod. Adds a ton of utilities and improvements to the level editor, such as redesigned menus, bug fixes, keybinds, and more!

    Installation Guide
    1. Install Geode by following the instructions at https://geode-sdk.org/download.
    2. Open Geometry Dash. You should see the Geode mod loader interface in-game.
    3. Click the "Mods" button in the Geode interface.
    4. Search for "BetterEdit" in the mod browser.
    5. Click "Install" on the BetterEdit mod entry.
    6. Restart Geometry Dash to activate BetterEdit.
